## 冒泡排序(Bubble)

### 算法描述

1.双层循环，两两比较，把大的数值换到后面

```js
const bubbleSort = (nums) {
  const n = nums.length
  for (let i = 0; i < n - 1; i++) {
    for (let j = 0; j < n - 1 - i; j++) {
      if (nums[j] > nums[j + 1]) {
        const t = nums[j]
        nums[j] = nums[j + 1]
        nums[j + 1] = t
      }
    }
  }
  return nums
}
```
