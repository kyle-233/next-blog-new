## 准备

-   github
-   vercel
-   code

## 开始

### 1、创建仓库

在[github](https://github.com)中创建一个仓库，根据提示把前一次的代码提交到仓库

### 2、部署

在[vercel](https://vercel.com)中创建一个项目，选择要部署的代码仓库。

> 注意： 如果没有看到代码仓库，可以在[github Applications](https://github.com/settings/installations)中授权仓库给 vercel
