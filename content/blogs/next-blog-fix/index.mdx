## 准备

-   Node.js ( >=18.18 version )
-   npm

## 开始

### 表格

```shell
npm i remark-gfm
```

```tsx title="mdx-component.tsx" {5}
import remarkGfm from 'remark-gfm'
const options = {
	parseFrontmatter: true,
	mdxOptions: {
		remarkPlugins: [remarkGfm],
		rehypePlugins: [
			rehypeKatex,
			[
				rehypePrettyCode,
				{
					theme: 'material-theme-palenight',
				},
			],
		],
	},
}
```

### `Link`

```css title="mdx.css"
.markdown a {
	@apply border-b-[1px] border-(--link) text-(--link);
}
```

### `tailwind.config.ts`

删除`tailwind.config.ts`文件
